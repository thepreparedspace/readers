word set

By default, the child's own carousel loops through Set 1, then Set 2, then back to Set 1 again — automatically, with no need to come back to you. Both sets are fixed and fully vetted for decodability. Shuffle is a separate, adult-only option — useful for review or a lesson you're actively guiding — and draws a fresh random ten from the whole pool each time you select it. Note: th has a smaller vetted pool than sh/ch, so it only has one set to loop through.

how to present this work

Tap the small name-slip at the top of each section first — the child hears the one sound alone, exactly as a sandpaper letter is isolated before it appears in a word. Each word card below shows the whole word immediately, ending in red. The child reads it using sounds already known. Tapping any single letter or the red ending plays a real recorded human voice — not synthesized speech — for that sound, useful when a child is unsure of a particular letter, especially ones often confused, like b, d, and g. Tapping the word itself reveals a speaker icon — tapping that speaks the whole word naturally, as a control of error the child can use on their own, without needing you to confirm it.

keeping it phonetic

Every word here is built only from sounds the child already has, apart from the target ending — there are no puzzle words in this set.

a note on "ng" and "nk"

These aren't quite the same kind of pattern as a true digraph like "sh" — "ng" is two letters fusing into one new sound, but "nk" is really the letter "n" quietly borrowing that same sound right before "k". Both are shown the same way here (the ending in red) for consistency, and this reader keeps to that simpler model rather than also marking the small shift in the vowel sound just before each ending — a further refinement worth introducing later, once this stage is secure.

extending to the home

This reader is meant as an extension of the prepared environment into the home — usable by a parent unfamiliar with Montessori method, so the child can keep practicing independently between sessions, supporting a smooth transition into elementary reading.
